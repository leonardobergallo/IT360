# IT360 - Soluciones Informáticas para Pymes y Profesionales

Sitio web profesional para IT360, empresa de soluciones informáticas integrales.

## 🚀 Características

- **Landing page** atractiva con hero section y servicios destacados
- **Catálogo de productos** con filtros por categoría
- **Página de servicios** con información detallada
- **Formulario de contacto** con integración WhatsApp
- **Diseño responsive** optimizado para móviles
- **SEO optimizado** para captar leads locales

## 🛠️ Tecnologías

- **Frontend:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Base de datos:** PostgreSQL con Prisma
- **Autenticación:** NextAuth.js
- **Deployment:** Vercel

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/leonardobergallo/IT360.git
cd IT360

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env.local

# Ejecutar en desarrollo
npm run dev
```

## 🌐 Deploy

El proyecto está configurado para deploy automático en Vercel.
